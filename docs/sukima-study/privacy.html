1. 本ポリシーについて

本プライバシーポリシー（以下「本ポリシー」）は、スマートフォンアプリ「スキマ学習」（以下「本アプリ」）におけるユーザーの個人情報およびデータの取り扱いについて説明します。本アプリは、忙しい社会人がスキマ時間に5分間の学習セッションを通じて知識を定着させることを支援するサービスです。

本アプリをご利用いただくことにより、本ポリシーの内容にご同意いただいたものとみなします。本ポリシーは、別途公開している「利用規約」と併せてご確認ください。

2. 収集する情報

2-1. サーバーに保存される情報

本アプリは、以下の情報をSupabase（データベース）に保存します。

情報の種類 内容 収集タイミング
認証情報 Apple ID またはGoogle アカウントの識別子（メールアドレス相当） サインアップ時
匿名ユーザーID 登録前のお試し利用で発行される一時的なID 「まず1回試す」タップ時
学習進捗 コースの受講状況・現在カード位置・完走記録 学習セッション中
学習成果 XP・ストリーク日数・最終学習日・クイズ回答履歴 セッション完了時
AI生成テーマ コース生成時にユーザーが入力したキーワードテキスト コース生成時
クォータ情報 AI生成の残回数・リセット予定日 生成時・月次

2-2. 端末内のみに保存される情報（サーバー非送信）

情報の種類 内容
通知時刻設定 学習リマインダーの時刻（例：21:00）
アプリ操作フラグ オンボーディング完了・スワイプヒント表示済みなどのフラグ
テーマ設定 ダーク/ライトモードの設定

2-3. 収集しない情報

本アプリは以下の情報を収集しません。

  • 位置情報（GPS）
  • カメラ・マイク・連絡先
  • 広告ID（IDFA等）
  • ブラウジング履歴・行動ログ

3. 情報の利用目的

収集した情報は以下の目的にのみ使用します。

  • 学習コースおよびカード、クイズの提供
  • AIによるコース生成（ユーザーが入力したテーマに基づく）
  • 学習進捗・XP・ストリークの記録と表示
  • AI生成クォータの管理
  • サービスの改善・不具合対応

4. 第三者サービスへのデータ送信

4-1. Supabase（データベース・認証）

本アプリのユーザーデータはSupabaseが提供するクラウドデータベースに保存されます。サーバーは東京リージョン（ap-northeast-1）に所在します。

Supabaseのプライバシーポリシーは https://supabase.com/privacy をご参照ください。

4-2. Google Gemini（AI生成）

ユーザーがAIコース生成機能を使用する場合、入力したテーマ（キーワードテキスト）がGoogle Gemini APIに送信されます。

内容
送信されるデータ ユーザーが入力したコースのテーマテキスト
送信されないデータ ユーザーID・メールアドレス等の個人識別情報
送信経路 本アプリ → Supabase Edge Function → Google Gemini API
通信 HTTPS（TLS）で暗号化

Googleのプライバシーポリシーは https://policies.google.com/privacy をご参照ください。

4-3. Apple / Google（OAuth認証）

サインイン時に使用するApple・Googleの認証情報は、各社のプライバシーポリシーに従って処理されます。本アプリはこれらの認証情報を直接保持しません。

4-4. Expo（OTA更新）

本アプリはExpoのOTA（Over-the-Air）アップデート機能を使用しています。アップデート確認時に、アプリのバージョン・ビルド番号・プラットフォーム（iOS/Android）・アップデートチャンネルがExpoのサーバーに送信されます。これらはユーザーを個人識別しない情報です。

ExpoのプライバシーポリシーはExpoの公式サイトをご参照ください。

4-5. RevenueCat（課金管理・プレミアムプラン）

プレミアムプランの課金処理にRevenueCatを使用します（Phase 2以降）。支払い情報はApp Store / Google Playが管理し、本アプリはクレジットカード番号等の支払い情報を保持しません。

5. データの保管と削除

5-1. 保管期間

アカウントが有効な期間中、学習データを保管します。アカウント削除後は速やかにサーバー上のデータを削除します。

5-2. ユーザーによるデータ削除

設定画面の「アカウントを削除する」から、いつでもアカウントおよびすべての学習データを削除できます。削除後のデータ復旧はできません。

5-3. 端末ローカルデータ

アプリをアンインストールすると、端末内に保存されたデータ（MMKV）はすべて削除されます。

6. セキュリティ

  • 外部APIとの通信はすべてHTTPS（TLS）で暗号化します
  • 端末内の認証情報はExpo SecureStore（iOS: Keychain、Android: Keystore）を使用して保護します
  • 本アプリはユーザーの同意なしに第三者にデータを売却・共有・提供しません

7. 広告について

本アプリは広告を表示しません。ユーザーの行動データが広告目的で使用されることはありません。

8. 年齢制限

本アプリは18歳以上を対象としています。18歳未満の方のご利用はご遠慮ください。

9. ユーザーの権利

ユーザーは以下の権利を有します。

  • アクセス権：自分のデータの内容を確認する権利
  • 削除権：アカウントおよびすべてのデータを削除する権利

これらの権利を行使したい場合は、下記「お問い合わせ」からご連絡ください。

10. ポリシーの変更

本ポリシーは予告なく更新される場合があります。重大な変更がある場合、アプリ内で通知します。最新版は本ページで常に確認できます。
