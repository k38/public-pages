第2条 サービスの定義と提供方法

2-1. 本サービスの内容

本サービスは、ユーザーが以下の機能を利用できるプラットフォームです：

  • 場所、食事、商品などの日常の体験を記録する機能
  • 記録に対して星評価（1〜5）とメモを追加する機能
  • 家族やペットを含む複数のメンバーを登録し、メンバー別に評価を行う機能
  • 記録の閲覧、検索、統計分析の機能
  • 有料の AI 分析機能（オプション）

2-2. 提供プラットフォーム

本サービスは iOS および Android デバイス上で Expo フレームワークを通じて提供されます。機能やUIは予告なく変更される可能性があります。

2-3. システム要件

ユーザーは本サービスを利用するために必要な互換性のあるデバイスおよびインターネット接続を自己の責任において準備するものとします。

第5条 データの保存とプライバシー

5-1. 端末内保存が基本

ユーザーが本アプリで記録したすべてのデータ（メモ、写真、評価、位置情報メタデータ）は、デフォルトではユーザーのデバイス上にのみ保存されます。当社はこのデータへのアクセスを有しません。

5-2. AI 分析機能の利用時のデータ送信

ユーザーが有料の AI 分析機能を利用する場合、メモテキスト内容と星評価（数値）のみが Google Gemini API に送信されます。

重要な注意：メモ欄に個人情報（他人の氏名、電話番号、住所、病歴など）や機密情報を入力した場合、それらの情報は匿名化処理されることなく AI に送信されます。

【AI分析機能をご利用の場合】

送信されるデータ：

  • メモのテキスト：ユーザーが手動で入力した感想・コメント
  • 評価値：1〜5の星評価（数値）

送信されないデータ：

  • GPS位置情報（端末から取得した位置座標）
  • 写真・画像ファイル
  • ユーザー識別情報（氏名、メールアドレス等）
  • 記録のメタデータ（訪問日時、場所名等）

送信先：

本アプリ → Cloudflare Workers（中継・API キー管理）→ Google Gemini API

Google Gemini APIのデータ処理契約（DPA）とSCC対応

本アプリから Google Gemini API に送信されるデータは、以下の国際的なデータ保護枠組みに基づいて処理されます：

  • データ処理契約（DPA）：Google は GDPR 第 28 条に基づくデータ処理契約に従い、当社からのデータを処理します（Google Cloud Data Processing Amendment）
  • SCC（Standard Contractual Clauses）：EU / EEA からのデータが米国の Google サーバーに転移される場合、SCC に基づき個人データは保護されます
  • Supplementary Measures：Google サーバーでの暗号化、ユーザーアクセスの制限、Google 従業員の厳格な秘密保持義務、定期的なセキュリティ監査

モデル学習における保護

  • 当社は Google Gemini API に対して「ユーザーデータ学習無効（UDLD）」設定を適用しています
  • 本アプリから送信されたメモテキストおよび評価データは、Google の機械学習モデルの学習に使用されません

国際的なデータ転移とEUユーザーへの通知

本アプリの開発者は日本に所在しており、Google サーバーは米国に所在しています。本アプリを EEA から利用する場合、メモテキスト・評価データは米国に転移されます。この転移は、GDPR 第 5 章（国際転移）に基づく SCC によって法的に保護されています。

📌【致命的リスク回避】自分自身の機密情報に関する警告

AI分析機能を利用する場合、以下の情報をメモに入力しないでください：

  • 健康関連情報：病名、診断結果、症状、医療治療履歴、薬の服用記録、メンタルヘルスに関する情報
  • 個人識別情報：パスポート番号、マイナンバー、運転免許証番号、銀行口座番号、クレジットカード番号、保険証番号
  • 思想・信条・信仰：宗教信仰、政治見解、イデオロギー、組合員資格、政党所属
  • 生物学的情報：遺伝子情報、DNA情報、バイオメトリクス
  • その他機密情報：犯罪歴、逮捕歴、裁判履歴、その他高度に個人的な機密情報

これらの情報が Google Gemini API に送信された場合、米国のサーバーに保存される可能性があり、高いプライバシーリスクが発生します。当社は、ユーザーが自発的に入力し、Google に送信された一切の損害について責任を負いません。

ユーザーのリスク責任

以下の場合について、当社は責任を負いません：

  • ユーザーがメモ欄に第三者の個人情報を入力し、AI に送信された場合
  • ユーザーが自分自身の機密情報・要配慮個人情報を入力し、Google Gemini API に送信された場合
  • その結果生じた個人情報の流出、プライバシー侵害、身元詐欺等による損害

5-3. AI 送信による個人情報の漏洩リスク

ユーザーがメモ欄に第三者の個人情報や機密情報を入力し、AI 分析機能を利用することで生じた一切の損害について、当社は責任を負いません。ユーザーは AI 分析機能を利用する前に、メモ内容を十分に確認するものとします。

5-4. 位置情報について

  • ユーザーがデバイスの位置情報利用を明示的に許可した場合にのみ使用
  • 位置情報は Open Street Map Overpass API（読み取り専用）に送信され、付近の施設名を取得するためにのみ使用される
  • 位置情報そのものや個人識別情報は、当社サーバーに保存されない

5-5. 写真について

  • 写真はデバイス内にのみ保存される
  • AI 分析機能を利用する場合、写真は送信されない
  • ユーザーは任意でアプリ内から写真を削除できる

5-6. 第三者 API との連携

Open Street Map (OSM) Overpass API

  • 目的：ユーザーの現在地付近の施設名を取得
  • 送信データ：GPS 座標（緯度経度）のみ
  • プライバシー：OpenStreetMap は個人情報を収集しない
  • ライセンス表示：本アプリの設定画面および「About」表示に「© OpenStreetMap contributors」と表示

Open Food Facts API

  • 目的：商品の JANコード（バーコード）から商品名、ブランド情報を取得
  • 送信データ：JANコード（バーコード番号）のみ
  • プライバシー：個人情報を含まない
  • ライセンス：Open Food Facts は Open Database License (ODbL) に準拠

Google Gemini API（AI 分析・有料機能）

  • 目的：記録内容の分析、インサイト生成
  • 送信データ：メモテキスト、星評価のみ
  • 経由：Cloudflare Workers

5-7. Cloudflare Workers について

Google Gemini API を直接呼び出さないために、Cloudflare Workers を経由してリクエストを送信します。以下の原則に従います：

  • API キーはサーバー側で管理され、アプリに埋め込まれていません
  • Cloudflare Workers は Cloudflare のプライバシーポリシーに準拠します（Cloudflare Privacy Policy）
  • Cloudflare は GDPR、CCPA、その他の個人情報保護規制に準拠しています（Cloudflare Trust Hub）

5-8. データ削除

  • ユーザーは本アプリをいつでも削除できます
  • アプリの削除後、デバイス内に保存されていたデータは復旧できません

5-8b. OS標準バックアップとの区別について

本アプリは外部送信を行いませんが、以下の点にご注意ください：

  • 本アプリのデータは端末内にのみ保存されます
  • ただし、iOS/Android の標準バックアップ機能が有効な場合、OS が自動的に iCloud / Google Drive / Google One にバックアップを取得する可能性があります

これは本アプリの機能ではなく、OS の仕様です。

バックアップを望まない場合：

iOS: 設定 → [ユーザー名] → iCloud → バックアップ → この iPhone からバックアップ を OFF
Android: 設定 → Google → アカウントサービス → バックアップと復元 を OFF

第6条 AI 分析機能と課金

6-1. AI 分析機能の概要

AI 分析機能（以下「AI プラン」）は、ユーザーの記録をもとに月間の行動パターンを分析し、ユーザーの好みや傾向をインサイト形式で提示し、パーソナライズされたおすすめを生成する有料機能です。

6-2. 課金モデル

AI プランは月額課金（サブスクリプション型）です。詳細な金額と内容は本アプリの「AI プラン」セクションに表示されます。

6-3. 自動更新と解約

  • AI プランの購読期間が終了する 24 時間前までに解約しない限り、購読は自動的に更新されます
  • ユーザーは毎月の更新日の前にいつでも解約できます
  • 解約の手続きはデバイスの App Store / Google Play 設定から行えます
  • 解約後、次の請求日に課金は発生しません

6-4. 課金プラットフォーム

  • iOS：App Store In-App Purchase
  • Android：Google Play Billing

ユーザーの支払い情報は App Store / Google Play によって管理され、当社はこれを保有しません。

6-5. 無料トライアル

初回利用時は限定的なトライアルアクセスを提供する可能性があります。詳細はアプリ内で告知されます。

6-6. 返金について

購読キャンセル後の返金は App Store / Google Play の規約に従います。
